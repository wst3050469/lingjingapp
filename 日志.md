# 灵境开发日志

## v1.73.141 - APK闪退修复 + 下载页修复 + 版本更新 (2026-06-21 15:00)

### 问题与修复

#### 问题1: 下载页只显示Android
- **根因**: versions.json中v1.73.140的`files`字段只有`android`
- **修复**: 补全5平台条目(win-x64_setup, win-x64_portable, linux-x64, linux-x64_deb, android)

#### 问题2: APK安装后闪退
- **根因**: 旧APK构建可能缺少babel/gesture-handler修复
- **修复**: gradlew assembleRelease重构建→部署为v1.73.141
- **版本递增**: 1.73.140→1.73.141, versionCode: 67→68
- **新APK**: 36.4MB, sha512: f1579c66b09eeeaa47f605930dffaef305d2b8c00aa80787de4ecd4e527d425da

#### 问题3: cloud-server versions.json不同步
- **根因**: /root/cloud-server/versions.json停在v1.73.120
- **修复**: 同步到最新+重启PM2

### 关键教训
1. 部署新版本必须同步3处versions.json
2. versions.json的files字段决定downloads.js展示哪些平台
3. APK闪退: babel.config.js需reanimated/plugin + App.tsx需gesture-handler导入
4. 修复bug后必须更新版本号(用户才能收到更新推送)
